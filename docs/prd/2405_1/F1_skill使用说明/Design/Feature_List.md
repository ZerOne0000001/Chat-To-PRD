# 功能清单梳理 (Feature List)

| 一级模块 | 二级功能 | 功能描述 | 优先级 |
| :--- | :--- | :--- | :--- |
| **全局框架** | 全局进度导航器 | 页面顶部或侧边悬浮的步骤条（Stepper），展示当前大纲的完整阶段序列，并高亮当前处于的阶段。 | P0 |
| | 快速跳转 | 演示者点击导航器上的任意阶段，页面主体内容瞬间平滑切换/滑入该阶段的卡片，实现快速前后文回顾。 | P0 |
| | 场景分支动态加载 | 当用户选择不同的场景（标准工作流/快速通道）时，全局导航器和主体卡片会动态刷新，隐藏不相关的阶段（例如快速通道会隐藏 1~3 阶段及 6 阶段）。 | P1 |
| **技能简介区** | 技能基础信息 | 展示 `pm-workflow` 的名称及一句话简介，作为大纲页面的入口。 | P0 |
| | 触发条件 | 列出该技能常见的唤起指令或触发条件（如：输入一句话需求、提供原型文件）。 | P0 |
| **场景选择区** | 场景卡片切换 | 提供“标准工作流”和“快速通道”两个选项卡，用户点击后切换下方大纲和导航器的内容。 | P1 |
| **阶段内容区** | 阶段大卡片 (Slide) | 每次仅聚焦展示一个阶段的详细内容，采用类似 PPT 幻灯片的设计，信息极度纯净。 | P0 |
| | 阶段动作简介 | 简要描述该阶段 AI 将执行什么核心动作（如：系统调研、业务流程图绘制）。 | P1 |
| | 阶段产物列表 | 列出该阶段会生成的具体的 Markdown 或 HTML 文件名称，并附带一句简短介绍（不需要截图或缩略图）。 | P0 |
